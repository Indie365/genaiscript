// metadata and model configuration
script({ model: "gpt4", temperature: 0 })

## Features

GenAIScript brings essential LLM prompt tooling into a cohesive scripting environment.

<CardGrid>

<Card title="Stylized Javascript" icon="seti:javascript">

Build prompts programmatically using [a stylized JavaScript syntax](/genaiscript/reference/scripts/).

```js wrap
// $ writes the prompt text
$`Your are a helpful assistant. Today is ${new Date()}.`
```

</Card>

<Card title="Fast Development Loop" icon="rocket">

Edit, [debug](/genaiscript/getting-started/debugging-scripts/), [run](/genaiscript/getting-started/run-scripts/) your scripts in the [Visual Studio Code](/genaiscript/getting-started/installation).

![Debugging a script](../../assets/debugger.png)

</Card>

<Card title="Reuse and Share Scripts" icon="star">

Scripts are [files](/genaiscript/reference/scripts/)! They can be versioned, shared, forked, ...

<FileTree>

-   genaisrc
    -   my-script.genai.js
    -   another-great-script.genai.js

</FileTree>

</Card>

<Card title="Data Schemas" icon="list-format">

Define, use and validate structured data using [schemas](/genaiscript/reference/scripts/schemas).

```js wrap
defSchema("MY_DATA", { type: "array", items: { ... }, })
$`Extract data from files using MY_DATA schema.`
```

</Card>

<Card title="Ingest PDFs, DOCX, CSV, .." icon="seti:pdf">

Seamlessly ingest and manipulate [PDFs](/genaiscript/reference/scripts/pdfs), DOCX, CSV, and other file formats.

```js
const { file } = await parsers.PDF(env.files[0])
const { file } = await parsers.DOCX(env.files[1])
```

</Card>

<Card title="RAG built-in" icon="magnifier">

[Embeddings vector search](/genaiscript/reference/scripts/embedding-search) powered by [LLamaIndex](https://ts.llamaindex.ai/).

```js wrap
// embedding vector index and search
const { files } = await retreival.search("cats", env.files)
```

</Card>

<Card title="Automate" icon="github">

Automate your scripts using the [CLI](/genaiscript/reference/cli).

```bash frame="none" wrap
node .genaiscript/genaiscript.cjs run my-script "*.pdf"
```

</Card>

<Card title="LLM Composition" icon="external">

Use LLMs to build your LLM prompts.

```js wrap
// summarize each files individually
for (const file of env.files) {
    const { text } = await runPrompt(_ => {
        _.def("FILE", file)
        _.$`Summarize the FILE.`
    })
    // use result in main prompt
    _.def("SUMMARY", text)
}
// use summary
$`Summarize all the summaries.`
```

</Card>
